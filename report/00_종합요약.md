## 0. 연구 흐름

기존 스크립트는 *"위기에 GMV 비중을 무엇이 결정하나"* 를 물었지만, 방법론 감사 결과 그 설계가 **순환적이고 누수가 있어** 결과를 신뢰할 수 없었습니다. 질문을 세 번 재설계하며 각 결함을 고쳤고, 선행연구에 정합하는 깨끗한 결과 + 위기 케이스 스터디로 마무리했습니다.

```
감사 (CRITICAL 3개)
   │
   ├─ 연구 1  Δw 위기 비중변화 (DiD)        → 평균회귀가 지배, 신호 약함
   │
   ├─ 연구 2  비중 수준 설명 (LW 전용)      → GMV = 저베타 임계 포트폴리오  ★ 헤드라인
   │
   └─ 연구 3  위기 케이스 스터디 (VIX 정의)  → 상대적 디리스킹, 절대 β는 압축
```

## 1. 감사 — 기존 설계의 문제

| # | CRITICAL 결함 | 위치 | 해결 방법 |
|---|---|---|---|

## 2. 연구 1 — Δw 위기 비중변화 (DiD)
세부: [`03_위기_비중변화_Δw.md`](03_위기_비중변화_Δw.md)

타깃 `Δw = w_crisis − w_pre`, 사전결정 피처 + `w_pre` 통제, Leave-One-Crisis-Out OOS.

| 모델 | Gerber | LW | Sample |
|---|---|---|---|
| M0 `w_pre` | **0.381** | 0.173 | 0.209 |
| M1 +Σ | 0.365 | 0.143 | 0.181 |
| M2 +非Σ | 0.192 | −0.178 | −0.042 |
| M3 GBR | 0.268 | 0.106 | 0.093 |

**결론:** 위기 비중변화는 **출발 비중의 평균회귀**가 지배(`w_pre` 계수 −0.74~−0.93, p≈0). 견고한 특성 드라이버는 `downside_vol`·`momentum`뿐이며, **Σ-파생 변수는 통제 후 무의미 → C1 실증.** 어떤 피처도 M0 베이스라인을 OOS로 못 이김(정직한 약신호).

## 3. 연구 2 — 비중 수준 설명 ★ (헤드라인)
세부: [`01_GMV가중치_설명연구.md`](01_GMV가중치_설명연구.md)

LW 전용. 타깃 = GMV 비중 수준 `w`, 20개 연말 비중첩 스냅샷(1,921행), 非Σ 헤드라인 피처 + Σ 분리 벤치마크, Leave-One-Year-Out OOS.

| 모델 | 피처 | R² |
|---|---|---|
| B0 Σ-벤치 | total_var, syst_share | 0.059 |
| M1 非Σ | beta, size, 유동성, 모멘텀 | 0.172 |
| M2 +섹터 | + GICS | 0.176 |
| **M3 GBR** | M2, 비선형 | **0.356** |

**결론:**
- **GMV는 저베타 포트폴리오.** beta 압도(OLS −0.034, t=−11; SHAP mean|val| 2위의 3배).
- **beta 효과는 선형이 아니라 임계(cliff):** β≲0.6에서 강한 양의 비중, 급락 후 β≳1은 ~0 바닥. 이 비선형성이 **M3가 선형의 2배(0.18→0.36)인 이유.**
- **CST2011 "Minimum-Variance Portfolio Composition"과 거의 정확히 일치**(해석적 임계 베타) — 여기서 모델-프리로 실증 복원.
- 항등식(C1)은 **long-only 비중에선 약함**: Σ-벤치가 OOS R²≈0.06뿐(long-only 제약이 `w∝Σ⁻¹1`를 깸, JM2003).

## 4. 연구 3 — 위기 케이스 스터디 (VIX 정의)
세부: [`02_위기_케이스스터디.md`](02_위기_케이스스터디.md)

**위기 정의:** VIX 이중임계 히스테리시스(진입>30/종료<20, 최소 10일, 42거래일 미만 병합) → **8개 명명 에피소드**(`tables/vix_crisis_periods.csv`): GFC×2, Flash Crash/EU 2010, EU 부채 2011, 중국 2015, Volmageddon 2018, COVID 2020, 금리 2022.

**결론:**
- **F1 — 심각 위기만 집중:** Effective-N COVID 16.5→8.5, GFC-Bear 13.3→5.6; 경미한 위기는 거의 변화 없음, 2022는 오히려 분산.
- **F2 — 2층 베타 효과(헤드라인 뉘앙스):** 횡단면 **β-비중 기울기가 가팔라짐**(6/8 위기; COVID −0.040→−0.072) — GMV가 저베타로 *더* 쏠림(CST2011 임계 tightening). **그러나 절대 포트폴리오 β는 상승**(COVID 0.52→0.60) — 상관 급등으로 베타가 1로 **압축**(LS2001).
- **F3 — 방어주로 도피:** 수혜는 저β 종목(VZ, JNJ, SO, WMT, LMT).
- **F4 — 유동성 null:** amihud 계수가 일관되게 안 움직임 — flight-to-liquidity 신호 없음(정직한 null).

**강건성:** 임계 방식(고정 30/20 vs 분위수 90/60) → 8 vs 9개, 8개 동일. 시장 프록시(EW vs SPY) → β 방향 8개 전부 동일, Effective-N 불변. 결론 유지.

## 5. 헤드라인 결론

> **Ledoit-Wolf long-only GMV 포트폴리오는 구조적으로 임계 베타(Clarke–de Silva–Thorley 2011)를 가진 저베타 포트폴리오이며, 본 연구에서 SHAP으로 실증 복원했다. 위기에는 저베타 틸트가 *강해지지만*(임계 tightening) 시장 전체 상관이 급등해 모든 베타가 1로 압축되므로(Longin–Solnik 2001) 실현 포트폴리오 베타는 오히려 *상승*한다. 심각한 충격(COVID·GFC)은 소수 방어주로 포트폴리오를 급격히 집중시킨다. 옛 순환적 설계의 초점이던 분산분해 변수들은 항등식·누수를 제거하면 설명력이 거의 없다.**

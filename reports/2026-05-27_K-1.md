## 1. 실험 설계

### 1.1 핵심 질문

Model K에 섹터 간 평균 상관(`cross_corr`)을 추가(Model K-1)하면 설명력이 개선되는가?  
Model M(total_var + within_corr + cross_corr)에서 cross_corr의 강한 유의성(평균 t=-3.58***)이  
syst_share·corr_min과 함께 투입해도 유지되는지 확인한다.

## 4. 핵심 발견

1. **K-1 핵심 결과**: cross_corr 추가 시 adj-R² +0.007 (0.198 → 0.204). cross_corr 평균 t=-0.45, 유의 창 비율 22.0%. syst_share·corr_min과의 상관으로 인해 cross_corr 독립 효과가 약화된다.

2. **corr_min 안정성**: K에서 평균 t=-2.39** (유의 창 72.5%) → K-1에서 평균 t=-1.97** (유의 창 63.3%). cross_corr 추가 후에도 corr_min 유의성이 유지 — 두 변수는 독립적 정보를 담는다.

3. **syst_share 안정성**: K 평균 t=-2.72*** → K-1 평균 t=-0.51. cross_corr 추가 후에도 부호·유의성 유지.

4. **cross_corr 맥락 의존성**: Model M(total_var 맥락)에서 평균 t=-3.58*** → Model K-1(syst_share·corr_min 맥락)에서 평균 t=-0.45. cross_corr 효과가 맥락 의존적으로 약화 — syst_share가 섹터 간 상관 정보의 일부를 흡수.

5. **전체 최선 모형**: Model F — 평균 adj-R²=0.286. K-1(0.204)은 F(0.286)에 비해 -0.082.

## 6. 한계

1. **선택 편의**: cross_corr은 Model M 결과를 보고 투입 — in-sample 확인 성격.
2. **다중공선성**: avg_corr / cross_corr / corr_min은 상호 상관이 높을 수 있어 VIF 검토 필요.
3. **비제약 GMV**: 음수 비중 허용 — 실제 운용 제약과 다를 수 있음.
4. **GFC 제외**: 2015-2024 데이터에 GFC(2007-2009)가 없어 극단 위기 행동 미확인.
